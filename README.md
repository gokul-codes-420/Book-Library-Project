# 📚 Noolagam - Book Library Project

A clean, minimalist, and beautifully designed web application for a Tamil Literature Book Library.

## 🚀 Live Demo & Local Preview

**[🟢 Live Preview: Click here to view the Book Library!](https://gokul-codes-420.github.io/book-library-live/)**

*(Note: It may take 1-2 minutes for GitHub Pages to build the site if you just clicked it right after deployment. If you see a 404, just refresh in a moment!)*

**[💻 Local Preview: Click here to open locally](./index.html)**

## 📁 Features
* **Elegant UI:** A sleek, light-themed aesthetic focused on minimalist typography (Inter & Lora fonts).
* **Interactive Cart:** Add items to your bag, see total calculations in real-time.
* **Smart Search:** Instantly filter books by title, author, or genre using the search bar.
* **Modern & Responsive:** Carefully crafted with pure HTML, CSS, and Vanilla JavaScript.

## 💻 Local Setup
To run this project locally:
1. Open the project folder in VS Code or your preferred terminal.
2. If using Python, you can run a local server:
   ```bash
   python3 -m http.server 8000
   ```
3. Open `http://localhost:8000` in your browser. Or, simply use the "Live Server" extension in VS Code.
